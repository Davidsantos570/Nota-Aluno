# 🏫 Sistema de Médias Escolares (Java)

Este projeto foi desenvolvido para consolidar meus conhecimentos em **lógica de programação estruturada** e **precedência de operadores matemáticos** utilizando a linguagem Java.

## 📋 Sobre o Projeto
O programa simula um sistema de secretaria escolar que:
1. Recebe três notas de um aluno via console.
2. Calcula a média aritmética (garantindo a ordem correta das operações).
3. Classifica o aluno em três estados: **Aprovado**, **Reforço** ou **Reprovado**.

## 🚀 Funcionalidades
- **Entrada Dinâmica:** Uso da classe `Scanner` para interação em tempo real.
- **Lógica de Decisão:** Estruturas `if`, `else if` e `else` para fluxos complexos.
- **Cálculo Preciso:** Uso de parênteses para garantir que a soma das notas ocorra antes da divisão.

## 🧠 Competências Demonstradas
- **Raciocínio Lógico:** Tradução de regras de negócio (critérios de aprovação) em código funcional.
- **Atenção aos Detalhes:** Correção de bugs de lógica e tratamento de tipos de dados (`double`).
- **Gestão de Processos:** Organização de arquivos e documentação clara do código.

## 🛠️ Tecnologias
- Java JDK 17+
- IDE (VS Code / IntelliJ)
